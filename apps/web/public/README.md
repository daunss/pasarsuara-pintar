# Public Assets

## Logo

Simpan file logo aplikasi di sini dengan nama `logo.png`

### Spesifikasi Logo:
- Format: PNG (dengan background transparan) atau SVG
- Ukuran: Minimal 200x200px
- Rasio: Square (1:1) atau sesuai desain logo
- File: `logo.png` atau `logo.svg`

### Desain Logo:
Logo aplikasi PasarSuara Pintar menampilkan:
- Topi oranye (melambangkan pedagang/UMKM)
- Chat bubble biru (melambangkan komunikasi via WhatsApp)
- Desain modern dan friendly

Logo ini akan digunakan di:
- Header aplikasi
- Halaman login/register
- Favicon browser
- Splash screen (jika ada)
